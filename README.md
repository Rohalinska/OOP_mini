# RealEstateAgency – Система управління нерухомістю

**Варіант №14** Підсумковий капстоун-проєкт у межах курсу ООП.

## Огляд проєкту
Цей застосунок призначений для автоматизації роботи агентства нерухомості. Він дозволяє керувати каталогом об'єктів (квартири, будинки), розраховувати комісійні та податки, а також проводити пошук нерухомості.

Проєкт реалізується ітеративно протягом 4-х лабораторних робіт.

## Архітектура
Проєкт побудований за принципами **Layered Architecture** (Багатошарова архітектура):
- **Domain:** Ядро системи, містить сутності (`Property`, `Apartment`, `House`) та бізнес-правила.
- **Application:** Сервіси для координації дій користувача.
- **Infrastructure:** Реалізація збереження даних (наразі In-Memory, далі — JSON).
- **Console:** Інтерфейс користувача.

## Поточний статус (Ітерація 1 - Lab 34)
На першому етапі реалізовано:
- Базову ієрархію класів із використанням абстракції та поліморфізму.
- Валідацію даних у конструкторах (захист від від'ємних цін).
- Юніт-тестування логіки розрахунку комісії.
- Налаштовано CI (GitHub Actions) для автоматичного збирання проєкту.

## Як запустити
1. Переконайтеся, що у вас встановлено **.NET 8 SDK**.
2. Клонуйте репозиторій.
3. Перейдіть у корінь проєкту та виконайте:
   ```bash
   dotnet restore
   dotnet build
   dotnet run --project src/RealEstate.Console
   ```
