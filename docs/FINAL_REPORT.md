# Фінальний звіт по проєкту Real Estate Agency

## Результати проєкту
Проєкт реалізує систему управління нерухомістю з можливістю збереження даних, пошуку та аналітики. 

## Використані технології
* .NET 8.0 / C#
* xUnit / Moq / Coverlet
* Git (гілки для кожної ітерації)

## Відповіді на контрольні питання (Lab 37)
1. **Рефакторинг перед релізом:** Найважливішим було додавання XML-коментарів та очищення `Program.cs` від зайвої логіки.
2. **Різниця README та FINAL_REPORT:** README - це "обличчя" проєкту для користувача, а FINAL_REPORT - це технічний аналіз для замовника/викладача.
3. **Кращі теми:** Найкраще продемонстровано SOLID, LINQ та Unit-тестування.
4. **Що б змінили:** Якби починали знову, використали б БД (SQLite) замість JSON для кращої цілісності даних.
5. **Готовність до захисту:** Проєкт має 80%+ покриття тестами, успішний CI-pipeline та повну документацію.
