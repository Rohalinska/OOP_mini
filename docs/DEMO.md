# Сценарій демонстрації (Demo Script)

1. **Запуск:** Показати порожній (або з 1-2 записами) файл `database.json`.
2. **Додавання:** Запустити додаток, додати нову квартиру (Адреса: "Незалежності 1", Ціна: 50000).
3. **Валідація:** Спробувати додати квартиру з ціною `-5` (продемонструвати обробку помилки).
4. **Аналітика:** Викликати пункт меню "Статистика" - показати, як LINQ порахував загальну суму.
5. **Persistence:** Закрити програму, відкрити знову - показати, що дані збереглися.
6. **Тести:** Запустити `dotnet test` у терміналі, показати, що всі 28 тестів пройдені.
