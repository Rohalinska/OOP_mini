# Матриця покриття тем курсу (Syllabus Coverage)

| Група тем | Що реалізовано в проєкті |
| :--- | :--- |
| **Основи ООП** | Класи `Property`, `Apartment`, інкапсуляція полів, конструктори з валідацією. |
| **Поліморфізм** | Перевизначення методів, використання базового класу `Property` для колекцій. |
| **Generics & Колекції** | Використання `List<T>`, `Dictionary<K,V>` для аналітики. |
| **LINQ** | Фільтрація (`Where`), агрегація (`Sum`), сортування (`OrderBy`), групування (`GroupBy`). |
| **Error Handling** | Власна ієрархія винятків (`InvalidPriceException` тощо), Fault handling у репозиторії. |
| **Persistence** | Асинхронна робота з файлами JSON через `System.Text.Json`. |
| **SOLID** | DIP (інверсія залежностей через інтерфейси), SRP (розділення сервісів та репозиторіїв). |
| **Патерни** | **Strategy** (для комісій), **Repository** (для даних). |
| **Тестування** | Unit-тести (xUnit), Mocks (Moq), Integration tests, Code Coverage. |
