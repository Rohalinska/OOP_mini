# Vision: Real Estate Management System

## Опис проблеми
Ріелтори витрачають багато часу на ручний розрахунок податків та перевірку доступності об'єктів. Проєкт автоматизує каталог нерухомості.

## Сценарії
1. Реєстрація квартири чи будинку.
2. Перегляд списку доступних об'єктів.
3. Розрахунок податку для кожного типу нерухомості.

## Нефункціональні вимоги
- Використання SOLID.
- Повна типізація даних.
- Наявність юніт-тестів.
